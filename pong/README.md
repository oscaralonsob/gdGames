# Pong
## Summary

The classic Pong game, a two-dimensional sports game that simulates table tennis.

The left paddle can be moved with the keys <kbd>W</kbd>, <kbd>A</kbd>, <kbd>S</kbd> and <kbd>D</kbd>.
